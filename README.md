## 📌 描述

对于高版本服主来说优化是个很头疼的问题

特别是村民更新之后对于TPS影响一直很大

这个插件是我针对自己服务器需求进行定制

不一定适用于全部的高版本服务器

大家可以根据使用情况来自行调整配置文件

如果遇到任何问题都可以评论留言我解决哦

## 🧩 功能介绍

### 🐾 stacker 生物堆叠器

开启后生物过于密集时堆叠

玩家可以看到生物堆叠数量

杀死他会得到更多的掉落物

当 TPS 高于设置值自动分离

### 🧱 spawner 刷怪笼限制

如果你服务器有很多刷怪笼

不妨可以试试开启这个选项

减少刷怪笼生成的生物速率

### 🐗 mob 生成生物限制

对于刷怪塔很有用

一个区块只能生成

你设定的生物数量

每天只能刷多少个

### 🔌 redstone 红石限制器

不对压力板或按钮开门有影响

TPS ≥ 16：不限任何红石

TPS < 16：限制高度 200 以上红石

TPS < 14：限制高度 150 以上红石

TPS < 12：限制高度 100 以上红石

TPS < 10：限制全部红石

### 🕹 observerPiston 限制活塞推动侦测器

对于高频活塞推动器进行清除

### ⏱ redstoneClock 高频限制器

对于高频红石进行清除

### ❄ snowball 防止雪球实体禁人塔

雪球应该在丢出后多久被清除

### ⛵ boatLimit 防止船实体禁人塔

限制船在区块内的数量

### 🧠 disableVillagerAI

当 TPS 太低时关闭村民 AI
